Satoshi Nakamoto

  • Started working on the code for Bitcoin
  • Registered the domain name bitcoin.org, and created a website at that address
  • Published a white paper describing a digital cryptocurrency, titled "Bitcoin: A Peer-to-Peer Electronic Cash System
  • Launched the Bitcoin network for the first time
  • Continued collaborating with developers, making all modifications to the Bitcoin source code himself
  • Gave control of the source code repository to Gavin Andresen, then disappeared
